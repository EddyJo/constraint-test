* 주문 정보를 전체 조회하고 주문정보를 조회하는 기능이 구현되어있습니다.
* 해당 기능에 대해 @SpringBootTest를 이용히여 테스트 케이스를 작성해주세요.
  * OrderInfoServiceTest는 BasicDbTest를 상속받았는데, 이미 BasicDbTest에 임베디드 Mysql 설정 및 @SpringBootTest이 완료되어 있습니다.
